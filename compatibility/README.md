# Mermaid compatibility evidence

This file is generated by `scripts/generate_compatibility.py`; do not edit it manually.
The repository tests Mermaid 11.16.0 as both the minimum and current latest supported 11.x release,
using Node 24.17.0 and separately locked toolchains. `swimlane-beta` sets the minimum version.
The Python distribution does not depend on Node, Mermaid CLI, Puppeteer, or Chromium.
